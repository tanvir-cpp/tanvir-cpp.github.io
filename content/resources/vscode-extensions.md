# Essential VS Code Extensions
Here is a curated list of VS Code extensions that I use daily:

1.  **ESLint** - Lints JavaScript and JSX.
2.  **Prettier** - Formats code.
3.  **Tailwind CSS IntelliSense** - Autocomplete for Tailwind.
4.  **GitLens** - Visualize git blame.
5.  **Auto Close Tag** - Automatically adds HTML/XML close tags.
6.  **Auto Rename Tag** - Automatically renames paired HTML/XML tags.
7.  **Live Server** - Launches a local development server with live reload.
8.  **Path Intellisense** - Autocompletes filenames.
9.  **Docker** - Manage containers and images.
10. **Thunder Client** - Lightweight Rest API Client.
11. **Markdown All in One** - All-in-one markdown editing support.
12. **Better Comments** - Create more human-friendly comments.
13. **Error Lens** - Highlights errors and warnings in-line.
